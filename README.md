# cs2-qq-bot
## QQ机器人连接CS2服务器 运行环境：Python 3.10
- 主要功能为群友自助查询在线玩家，不止能查人数，私聊BOT还可以踢人，管理bot等，
- 可以执行私聊发给他的任何服务器命令（QQ命令可以包含空格添加参数了！）
- csgo\cfg\server.cfg文件里配置rcon密码：rcon_password ""
- 其他依赖及配置项请参照代码内介绍
### 依赖项目地址：
- https://github.com/Mrs4s/go-cqhttp
